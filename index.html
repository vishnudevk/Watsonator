IBM WATSON TRANSLATOR
& FOR EVERYONE

IBM Watson Translator app is available for free for everybody in a browser or on any smart phone. Our IBM Watson Translator will be able to translate the word 'once in a blue moon' correctly to 'very rarely' rather than giving a word-by-word translation, since our translator is able to learn by being trained.

How it will work: Users can use IBM Watson Translator in real-time by speaking/typing the words and phrases into a smartphone or a browser and asking Watson to translate it to the specific language desired (and be able to save the output). IBM Watson Translator will then return the translated words or phrases as text or speech as requested by the user.

How this can help improve the world: This app can help for example the refugees in Germany (or many other countries): Our Speech to Text app can understand Modern Standard Arabic, which can be translated into English (German currently not available) by the Language Translation app and if needed this translated text can then be transformed to speech by using our Text to Speech app. Syrian people can use the translator with all daily tasks while learning German/English language.
